📘 Подсказки

  • Делегирование событий — вешайте один обработчик на #board, а не на каждую кнопку карточки. Используйте e.target.closest('[data-action]') для определения нажатой кнопки.
  • e.stopPropagation() — останавливает всплытие события вверх по DOM-дереву. Нужен, когда клик на кнопку внутри карточки не должен «выделять» саму карточку.
  • e.target vs e.currentTarget — target — элемент, на котором произошло событие; currentTarget — элемент, на котором висит обработчик.
  • closest(selector) — ищет ближайшего предка (включая сам элемент), соответствующего селектору. Возвращает null, если не найден.
  • { once: true } — опция addEventListener: обработчик сработает ровно один раз и автоматически удалится.
  • removeEventListener — требует ту же именованную функцию, что передавалась в addEventListener. Анонимные стрелки удалить нельзя!
